# DDA Research Game — Cross-Platform Lane-Based

This is a **web-based research game** designed to study **Dynamic Difficulty Adjustment (DDA)** in games. Players interact with two versions of the game — Fixed Difficulty and Adaptive Difficulty — and provide feedback through a Google Form.

The game is built using **HTML, CSS, and JavaScript**, and works on both **PC and mobile devices**.

---

## Features

- **Lane-Based Gameplay**: Player moves left and right to avoid obstacles and collect items.
- **Two Modes**:
  - **Fixed Difficulty**: A constant level of challenge.
  - **Adaptive Difficulty**: Difficulty dynamically adjusts based on player performance.
- **Adaptive Locked Initially**: Players must complete the Fixed Difficulty round to unlock Adaptive.
- **Cross-Platform Controls**:
  - Keyboard support for PC (`Arrow Left`, `Arrow Right`)
  - Touch buttons for mobile devices
  - Swipe support for mobile
- **50 Interactions per Round**: Each game round contains 50 interactions.
- **Google Form Integration**: After playing Adaptive Difficulty, players are prompted to fill out a feedback form.

---

## How to Play

1. Open the game in a web browser (PC or mobile).
2. Start with **Fixed Difficulty**.
3. Complete 50 interactions (avoid red obstacles and collect yellow items).
4. After completing Fixed, Adaptive Difficulty is unlocked.
5. Complete Adaptive Difficulty round.
6. After finishing, click the button to fill out the [feedback form](https://docs.google.com/forms/d/e/1FAIpQLSd0Blfsm120BHedJyPrTHqoVjemJXaqKh5RzwBibhtGlRWc3A/viewform?usp=dialog).
